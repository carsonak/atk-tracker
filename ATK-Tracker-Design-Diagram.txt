+-----------------------------------------------------------------------+
|                         CLIENT AGENT LAYER                            |
+-----------------------------------------------------------------------+
|             Background Daemon (Go binary via systemd)                 |
|                                                                       |
|  - Startup Recon: Query D-Bus ListSessions for existing state         |
|  - Hardware Filter: Use ioctl bitmasks to find EV_KEY/EV_REL devices  |
|  - Catch OS Login/Logout events -> Init/End Session                   |
|  - Monitor HID (/dev/input) -> Calculate 1-sec active buckets         |
|  - Aggregate -> Sum active seconds over 5-min window                  |
|  - Local Buffer (SQLite/JSON) -> Store if network is offline          |
|  - Catch OS Suspend/Logout -> Flush buffer & End Session              |
